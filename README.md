1. ### 康师傅

   - **获取token**

     - 登录授权手动执行一次签到或抽奖，抓包相关链接
     - 签到：https://club.biqr.cn/api/signIn/integralSignInList
     - 抽奖：https://club.biqr.cn/api/game/turntable/open
     - 获取请求头部Token复制下来

   - 青龙变量

     - 增加青龙变量yml_ksf_data=TOKEN1@TOKEN2
     - 多账户@隔开
